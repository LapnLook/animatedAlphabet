The brief: create an animated alphabet using as few digital technologies as possible. The theme was "program or be programmed".

Instructions: type any of the 26 letters from the English alphabet and it will appear on the screen accompanied by a sound effect that is completely unique to it. Holding Shift or pressing Caps Lock enables alternate versions of each character. Space and Enter are the only other keys that function right now. You have 4 rows to work with, the number of columns is dependant on your browser window. Resizing the window erases the currently visible letters, but does not require a page refresh.
